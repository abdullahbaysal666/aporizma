Privacy Policy

Last updated: 2026-07-19 · Türkçesi aşağıdadır.

Aporizma web tools (aporizma.com)

Every tool on this site runs entirely in your browser. Files you drop and text you paste are processed locally with JavaScript and are never uploaded to any server. We do not use cookies, analytics scripts or trackers.

Aporizma browser extensions

Our extensions (such as Aporizma Translate) follow the same principle:

  • No data collection. We do not collect, store, transmit or sell any user data. We have no servers.
  • Your API key. If an extension uses a bring-your-own-key model, your key is stored only on your device (chrome.storage.local) and is sent only to the API provider you configured (e.g. Google's Gemini API), directly from your browser.
  • Your text. Text you translate travels directly from your browser to the API provider. It never passes through us.
