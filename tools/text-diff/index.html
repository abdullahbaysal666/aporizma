Paste both texts to compare.

Is my text uploaded?

No. The comparison runs entirely in your browser — safe for contracts, code and private notes.

How does the comparison work?

A longest-common-subsequence algorithm aligns the two texts line by line, so a single inserted paragraph doesn't mark everything below it as changed.

Can I compare code?

Yes — the diff is whitespace-exact and monospaced, so indentation changes are visible too.
