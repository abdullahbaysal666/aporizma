In what order are the options applied?

Trim, then remove blank lines, then remove duplicates, then sort, then reverse — so "Z → A" plus "Reverse" gives you A → Z again, and dedupe always runs before sorting.

Is duplicate removal case-sensitive?

Yes. "Apple" and "apple" are treated as different lines. Trim whitespace first if lines only differ by leading/trailing spaces.

Is my list uploaded anywhere?

No. Everything runs in your browser with JavaScript — nothing is sent, logged or stored. Sorting uses locale-aware comparison, so Turkish letters (ç, ğ, ı, ö, ş, ü) order correctly.
