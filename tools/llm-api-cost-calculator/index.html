How is this calculated?

Cost per request = (input tokens × input price + output tokens × output price) ÷ 1,000,000. Monthly cost assumes 30 days at your daily request volume.

What's not included?

Prompt-caching discounts (up to 90% on repeated context), batch API discounts (typically 50%), long-context surcharges, and free tiers. Real bills are often lower with caching — treat these numbers as a worst-case baseline.
