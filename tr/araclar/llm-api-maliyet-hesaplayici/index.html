LLM API Maliyet Hesaplayıcı

Tipik istek boyutunuzu ve günlük hacminizi girin — her büyük modelin istek başına ve aylık maliyetini yan yana görün.

Model Girdi $/M Çıktı $/M İstek başına Aylık

Fiyatlar 2026-07-19'da resmi sağlayıcı fiyat sayfalarından doğrulandı. Standart katman; önbellek/batch indirimleri hariç.

Nasıl hesaplanıyor?

İstek maliyeti = (girdi token × girdi fiyatı + çıktı token × çıktı fiyatı) ÷ 1.000.000. Aylık maliyet, günlük istek hacminizle 30 gün varsayar.

Neler dahil değil?

Prompt önbellek indirimleri (tekrarlanan bağlamda %90'a varan), batch API indirimleri (genelde %50), uzun-bağlam ek ücretleri ve ücretsiz katmanlar. Önbellekle gerçek faturalar çoğu zaman daha düşüktür — bu rakamları en-kötü-durum tabanı olarak okuyun.
